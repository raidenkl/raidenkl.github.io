在termux中Hexo-Admin的Deploy配置
发表于 2019-08-14| 更新于 2019-08-14|termux

1.进入hexo博客的文件夹，执行：

npm install --save hexo-admin

2.运行下列命令启动hexo-admin ：

hexo server -d

打开 http://localhost:4000/admin/ 就可以访问到hexo-admin管理页面了。

3.这里是重点，我自己就是没做这部导致Deploy配置不了。打开setting，点击Setup authentification here输入用户名，密码，密钥，下面会自动生成配置文件，复制

例：
admin:
username: xxx
password_hash: xxx
secret: xxx

复制图上箭头所指的部分复制就可以了。

4.Ctrl+c停止hexo博客，执行：

vim _config.yml

把刚才复制的东西粘贴到最下面，保存，退出。然后执行：

mkdir admin_script
cd admin_script
touch hexo-g.sh
vim hexo-g.sh

在里面添加：

#!/usr/bin/env sh
hexo clean
hexo g -d

保存后执行：

chmod +x hexo-g.sh

5.返回到hexo博客文件夹，打开_config.yml文件,在刚才复制的配置文件下面加上这句：

deployCommand: ./admin_script/hexo-g.sh

不懂看图：

注意：要像图上那样文字变了颜色才算成功，没变色的话多试几遍。

6.现在执行：

hexo s

再进入管理界面，点击Deploy，就可以一键部署了
