前言写这个教程的原因是发现网上关于树莓派安装zerotier都是用的官方脚本，但是官方脚本不能辨认kali-rolling，所以用编译的方法安装 1.下载源码到zerotier的github下载：点此下载,点击code->download zip，下载完成后解压： unzip ZeroTierOne-master.zip 解压完成后进入文件夹： cd ZeroTierOne-master 2.安装编译之后安装： sudo make && sudo make install 等待安装完成（大约15分钟） 3.使用启动主程序： sudo zerotier-one -d 加入到创建的网络： zerotier-cli join your_network_id 安装完成！ *.让zerotier开机自启参考我的上一篇文章
